ICÔNES POUR L'EXTENSION CHROME
================================

L'extension nécessite 3 icônes aux formats suivants :
- icon16.png  (16x16 pixels)
- icon48.png  (48x48 pixels)
- icon128.png (128x128 pixels)

COMMENT CRÉER LES ICÔNES :
--------------------------

Option 1 : Générateur en ligne
-------------------------------
1. Aller sur https://www.favicon-generator.org/
2. Uploader votre logo/image
3. Télécharger les icônes générées
4. Renommer les fichiers aux noms requis

Option 2 : Design manuel
-------------------------
1. Utiliser Figma, Photoshop, GIMP ou Canva
2. Créer 3 carrés aux dimensions mentionnées
3. Dessiner un logo simple (exemple : livre 📚 ou document 📄)
4. Exporter en PNG avec fond transparent

Option 3 : Utiliser un emoji
-----------------------------
1. Aller sur https://emoji-to-png.com/
2. Choisir l'emoji 📚 ou 📖
3. Générer aux 3 tailles
4. Télécharger et renommer

Option 4 : Placeholder simple
-------------------------------
Pour tester rapidement, créez un carré de couleur unie :
- Couleur : #2563eb (bleu)
- Texte : "SC" (SmartContext)

RECOMMANDATIONS DESIGN :
-----------------------
- Utiliser un design simple et reconnaissable
- Préférer un fond transparent
- Utiliser des couleurs contrastées
- Éviter les détails trop fins (illisibles à 16x16)

VÉRIFICATION :
--------------
Après avoir ajouté les icônes dans ce dossier (/extension/),
recharger l'extension dans Chrome pour voir les nouvelles icônes.

Si les icônes ne s'affichent pas, vérifier :
1. Noms de fichiers corrects (sensible à la casse)
2. Format PNG (pas JPG ou autre)
3. Permissions de lecture sur les fichiers
4. Chemins corrects dans manifest.json
