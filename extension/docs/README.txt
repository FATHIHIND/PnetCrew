PLACEHOLDERS PDF
================

Ce dossier contient les documents de démonstration pour SmartContext Doc.

Dans un environnement de production, vous devez remplacer ces fichiers par de vrais documents PDF.

Fichiers attendus :
- jira_guide.pdf : Documentation Jira
- teams_best_practices.pdf : Bonnes pratiques Microsoft Teams
- github_cheatsheet.pdf : Aide-mémoire GitHub
- azdo_quickstart.pdf : Guide de démarrage Azure DevOps
- confluence_guide.pdf : Guide Confluence

Pour tester l'extension sans avoir de vrais PDF, vous pouvez :
1. Créer des PDF factices en ligne (ex: https://smallpdf.com)
2. Utiliser des captures d'écran converties en PDF
3. Télécharger de vraies documentations officielles

Note: Le serveur Express peut servir ces fichiers statiquement via le endpoint /docs/
